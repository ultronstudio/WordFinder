Aplikace WordFinder je určená pro vyhledání uživatelem zadaného slova v textu.

## Informace
Aplikace WordFinder byla vytvořena jako domácí úkol z předmětu **PROGRAMOVÁNÍ** na [SPŠ Hradební](https://www.hradebni.cz)
